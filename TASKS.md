## Node

- Validate node
- Valid nodes can have different statuses

## Radiator

- Radiator can contain multiple nodes or none
- Validate existing nodes
- Radiator also has a number

## Space

- Has a name
- May have multiple radiators or none

## Unit

- Has a name
- May have spaces

## Floor

- Has a level
- Has a name
- May have spaces
- May have units
